# [1979. 找出数组的最大公约数 - 力扣（LeetCode）](https://leetcode.cn/problems/find-greatest-common-divisor-of-array/description/)

```java
class Solution {
    public int findGCD(int[] nums) {
        Arrays.sort(nums);
        int a = nums[0];
        int b = nums[nums.length - 1];
        while(b!=0){
            int t=a%b;
            a=b;
            b=t;
        }
        return a;
    }
}
```
